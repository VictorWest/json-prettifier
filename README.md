# JSON Prettifier (Lightweight)

A minimal Next.js app for formatting and beautifying JSON input.  
It provides real-time prettification, error fallback, and copy-to-clipboard functionality.

---

## Features
- Paste or type JSON into the left panel.
- Prettify JSON with proper indentation.
- Error handling via backend fallback.
- Copy formatted JSON with one click.
- Clean dark UI with social links.

---

## Tech Stack
- **Next.js** (React framework)
- **TypeScript**
- **Tailwind CSS**
- **Material UI**
- **React Icons**

---

## Installation
```bash
# Clone the repo
git clone <repository_url>

# Navigate to folder
cd json-prettifier

# Install dependencies
npm install

# Run development server
npm run dev
